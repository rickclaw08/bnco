**Apple Watch/HealthKit provides:**
- activeEnergyBurned, heartRateData (time-series BPM), workout duration/type

**New honest bnco Score (0-100):**
1. Effort Score (40%) - WHOOP strain mapped to 0-100, weighted by zone 2-4 time (productive Pilates zones)

3. Recovery Score (25%) - Direct WHOOP recovery_score mapping

## GHL HVAC Prompt Save - SOLVED (3:15 PM)
- **Root cause**: Setting textarea value via native HTMLTextAreaElement.prototype.value.set does NOT trigger Vue 3 reactivity (it's a Vue/Naive UI app). The Save button click goes through but Vue sees no state change, so no API call fires.
