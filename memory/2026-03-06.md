### Real Scoring Model (from WHOOP/Apple Watch API research)
**WHOOP Workout API provides:**
- strain (0-21, Borg Scale), average_heart_rate, max_heart_rate, kilojoule, zone_durations (6 HR zones)

**WHOOP Recovery API provides:**
- recovery_score (0-100%), resting_heart_rate, hrv_rmssd_milli, spo2_percentage, skin_temp_celsius

**Apple Watch/HealthKit provides:**
- activeEnergyBurned, heartRateData (time-series BPM), workout duration/type

**New honest bnco Score (0-100):**
1. Effort Score (40%) - WHOOP strain mapped to 0-100, weighted by zone 2-4 time (productive Pilates zones)
2. Consistency Score (35%) - Attendance streak + frequency multiplier (3x/wk = 1.0, 4x = 1.1, etc.)
3. Recovery Score (25%) - Direct WHOOP recovery_score mapping

**Key insight:** Apple Watch does NOT provide "Core Motion stillness" via API for third-party apps. Old "Stillness Index" claim was fabricated. Replaced with Consistency Score which is honestly trackable.
