### Key Technical Learnings
- GHL uses Tabulator grid library for contacts list
- Tabulator checkbox clicks via `input.click()` in evaluate don't propagate to GHL's selection state
- Only the header checkbox (col-title) triggers GHL's bulk selection mechanism
- Page size 20 = all 20 rows rendered, page size 100 = only 40 rows rendered (virtual scroll)
- Bulk delete requires typing "DELETE" in confirmation input, not just clicking
