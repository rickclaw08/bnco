## Recommended Screenshots (6 total)

### Screenshot 1: Hero / Landing
- Show the landing page with "Compete. Connect. Conquer." tagline
- Overlay text: "Your Pilates, Gamified."
- Highlight the athlete/studio count stats

### Screenshot 2: RES Score Dashboard
- Show the profile card with RES score, XP bar, streak info
- Overlay text: "Your Effort, One Score"
- Display the Pilates Vibe Score breakdown

### Screenshot 3: Live Leaderboard
- Show the class leaderboard with podium view
- Overlay text: "Compete in Real Time"
- Include the live indicator dot

### Screenshot 4: Ghost Racing
- Show the ghost racing timeline
- Overlay text: "Race Yesterday's Champion"
- Display the "You're ahead by 3.2 RES" status

### Screenshot 5: Studio Dashboard
- Show the studio owner view with stats and missions
- Overlay text: "Built for Studios"
- Display active members, avg score, city rank

### Screenshot 6: Achievements & Progression
- Show achievements grid and XP progression
- Overlay text: "Level Up Every Session"
- Display streak multiplier and badges

## Design Guidelines
- Use the BNCO color palette: sage green (#7C9082), cream (#F5F0EB), dark (#1A1A2E)
- Overlay text should use the Outfit font, weight 700
- Frame screenshots in device mockups (Apple provides templates)
- Keep text minimal - let the UI speak
- Ensure text is readable at small sizes

## Capture Process
1. Run the app on iPhone 15 Pro Max simulator (Xcode)
2. Navigate to each screen with sample data loaded
3. Capture via Cmd+S in Simulator or Screenshot tool
4. Add overlay text and device frames in Figma or similar
5. Export at exact required dimensions
6. Upload to App Store Connect under "App Information" > "Screenshots"
