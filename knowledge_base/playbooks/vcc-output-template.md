# VCC Output Template (Brand's Reference Design)

Created: 2026-03-01 14:15 EST
Source: Brand provided this as the gold-standard format for how VCC reports should look.

## Structure

### 1. VCC Initial Report Header
- Mode declaration (Architect / Mercenary / Hybrid)
- Lead Agents assigned with their specific objective
- Passive Agents listed with their Level-Up mode

### 2. ARCHITECT Section
- Mission name ("Project Octa-Scale" style)
- Strategic objective from the CEO role
- Includes specific directives to other agents
- Global Expansion Map or equivalent visual

### 3. MERCENARY Section
- High-velocity conversion asset (ad script, landing page, DM template)
- Must include: Hook, Body, Irresistible Offer, CTA
- Concrete numbers (48 hours, 3x efficiency, 5 firms per country)

### 4. Background Innovation Report
- What passive agents are doing while leads execute
- Each passive agent gets one bullet with a concrete deliverable
- Include measurable results ("reduces latency by 40%")

### 5. XP and Level-Up Summary
- Individual agent XP gains with specific skills
- File references for what was updated
- Company Level progression

## Example Scenario
"Scale a new AI software to 10 countries while maintaining $50k weekly profit"
- Hub and Spoke model: centralized AI logic, localized front-ends
- Localized pricing by purchasing power per market
- SLMs for data sovereignty compliance (Japan, Germany)
- Currency fluctuation stress testing (Brazil, Japan)
- Cultural nuance SOPs for copy localization
- Scarcity framing: "5 firms per country this month"
- 48-hour deploy + 30-day money-back guarantee structure

## Key Principles from This Template
- Every agent has a job, active or passive. No idle agents.
- Passive agents produce REAL deliverables, not status updates.
- XP logging creates permanent institutional knowledge.
- Reports are structured, scannable, and action-oriented.
