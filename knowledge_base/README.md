# Knowledge Base

This directory stores "XP" from agent missions. Each file represents learned intelligence that makes the company smarter.

## Structure
- `market/` - Market research, competitor intel, industry trends
- `playbooks/` - Proven strategies, templates, frameworks
- `tech/` - Technical learnings, architecture decisions, benchmarks
- `sales/` - Sales insights, objection handling, conversion data
- `ops/` - Operational learnings, process improvements

## XP Log Format
Each file should note:
- Which agent created it
- Date created
- What was learned
- How it applies to our business
