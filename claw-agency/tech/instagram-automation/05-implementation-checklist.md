## Risk Mitigation

| Risk | Mitigation |
|------|-----------|
| Account flagged for spam | Max 2 follow-up nudges. Never auto-DM without user initiating. |
| ManyChat outage | Monitor ManyChat status. Keep manual reply templates ready. |
| Low conversion rate | A/B test copy. Adjust timing. Improve content CTAs. |
| Keyword false positives | Use "contains" carefully. Monitor for unintended triggers. |
| Instagram API changes | ManyChat handles API updates. Stay on latest ManyChat version. |
