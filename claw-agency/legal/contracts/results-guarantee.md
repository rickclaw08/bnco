## 4. Conditions for Guarantee

This guarantee is valid only if the following conditions are met:

### Client Responsibilities
- **Adoption:** Automated workflows are actively used as designed (not bypassed or ignored)
- **Training:** Team members complete provided training and follow documented procedures
- **Environment:** No significant changes to systems, processes, or team size during measurement period
- **Cooperation:** Client provides requested data and participates in checkpoint reviews
- **Maintenance:** All third-party tools and integrations remain active and properly configured

### Exclusions
This guarantee does NOT apply if results are impacted by:
- Client's failure to adopt or properly use the automation
- Changes to Client's business processes, tools, or team structure
- Third-party platform outages, API changes, or deprecated features
- Force majeure events (natural disasters, pandemics, etc.)
- Unauthorized modifications to the automation by Client or third parties

## 5. Remediation Process

If guaranteed results are not achieved by the end of the measurement period:

### Step 1: Analysis (within 7 days)
Provider will conduct a root cause analysis to determine why targets were not met. Possible findings:
- Adoption issues (Client not using automation as designed)
- Technical issues (automation not performing as expected)
- Baseline issues (initial estimates were incorrect)
- External factors (third-party platform changes)

### Step 2: Remediation Proposal (within 7 days)
Provider will propose one of the following:

**Option A: Fix & Extend**  
Provider will make necessary improvements at no additional cost and extend the measurement period by [30/60] days.

**Option B: Alternative Solution**  
Provider will design and implement an alternative approach to achieve the guaranteed results.

**Option C: Partial Refund**  
If results were partially achieved, Client receives a prorated refund:
- 75-99% of target achieved = 25% refund
- 50-74% of target achieved = 50% refund
- 25-49% of target achieved = 75% refund
- <25% of target achieved = 100% refund

**Option D: Full Refund**  
If technical failure with no viable remediation path, Client receives 100% refund of project fee.

### Step 3: Client Choice (within 7 days)
Client selects preferred remediation option. If no response within 7 days, Provider defaults to Option A (Fix & Extend).
