## Job Posting 1: Freelance Python/Automation Developer

**Title:** Python Automation Developer for AI-First Agency (Remote, Flexible Hours)

**Job Type:** Contract, Part-Time to Full-Time (20-40 hrs/week)

**Hourly Rate:** $50-60/hr USD (negotiable for exceptional candidates)

---

### About ClawOps

We are an AI-first automation agency helping businesses eliminate manual work through intelligent workflows. We build custom integrations, data pipelines, and automation solutions using AI agents and Python. We are growing fast (targeting $100K revenue in 45 days) and need a talented developer to help us deliver world-class solutions.

### What You Will Do

**Core Responsibilities:**
- Build custom Python scripts for data scraping, API integrations, and workflow automation
- Work alongside AI agents (you will oversee/refine their code, not start from scratch)
- Debug and optimize existing automation workflows
- Integrate third-party APIs (Stripe, Airtable, Zapier, webhooks, etc.)
- Write clean, documented, maintainable code
- Collaborate with clients (via our PM) to understand technical requirements

**Example Projects:**
- Build a web scraper to collect competitor pricing data daily
- Create API integrations between CRM and accounting software
- Automate report generation from multiple data sources
- Build custom Slack/Discord bots for client workflows

### Who You Are

**Required:**
- 3+ years professional Python experience
- Strong understanding of APIs, webhooks, data formats (JSON/CSV)
- Experience with web scraping (BeautifulSoup, Selenium, Playwright)
- Comfortable reading/modifying others' code (including AI-generated code)
- Self-directed: you see a problem and fix it without waiting for instructions
- Excellent written communication (async-first work environment)
- Available for at least 20 hrs/week, flexible scheduling

**Bonus Points:**
- Experience with automation tools (n8n, Make, Zapier)
- Database skills (SQL, PostgreSQL, MongoDB)
- Cloud platform experience (AWS Lambda, Google Cloud Functions)
- Familiarity with AI/LLM APIs (OpenAI, Anthropic)
- Previous freelance/agency experience

**Not Required:**
- You do NOT need to be a senior architect or design complex systems
- You do NOT need to manage clients (we handle that)
- You do NOT need ML/data science expertise

### Why Work With Us

**Pros:**
- Work with cutting-edge AI tools (you will learn a ton)
- Flexible hours, fully remote
- Interesting projects (no boring CRUD apps)
- Fast-paced environment (see your work ship same week)
- Paid weekly, clear scope, respectful deadlines
- Potential for long-term partnership as we scale

**Honest Cons:**
- Startup pace (things change quickly)
- You will work alongside AI (if that bothers you, we are not a fit)
- Some weeks may be slower (contractor flexibility)

### How to Apply

**Include in your proposal:**
1. **Portfolio:** Link to 2-3 relevant projects (GitHub, live demos, or descriptions)
2. **Availability:** Hours/week you can commit, timezone
3. **Rate:** Your hourly rate (we will discuss if there is a fit)
4. **Quick Question:** Describe a time you automated something tedious. What tool did you use? How long did it take?

**We will review applications within 24 hours.** Strong candidates will receive a short skills test (paid, 2-3 hours). Final step is a 30-min video call.
