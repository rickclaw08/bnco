# ClawOps Contractor Hiring Plan

## Revenue Triggers & Hiring Timeline

### Phase 1: $0-25K (Weeks 1-2)
**Status:** AI-only operations
- No contractors needed yet
- Focus: Landing first 3-5 clients
- Blake (CEO) handles sales + delivery with AI agents

### Phase 2: $25K-50K (Weeks 3-4)
**Trigger:** 5+ active clients OR 1 enterprise client
**First Hire:** Freelance Python/Automation Developer (20-40 hrs/week)
- **Why:** AI agents need human oversight for complex custom integrations
- **When:** When we have 2+ concurrent client projects requiring custom code
- **Cost:** $30-60/hr = $2,400-9,600/month
- **ROI:** Frees Blake to sell more, increases delivery capacity 3x

### Phase 3: $50K-75K (Weeks 5-6)
**Trigger:** 10+ active clients OR complexity overload
**Second Hire:** Virtual Assistant/Project Coordinator (30-40 hrs/week)
- **Why:** Client communication, scheduling, project tracking at scale
- **When:** Blake spending >10 hrs/week on admin tasks
- **Cost:** $20-35/hr = $2,400-5,600/month
- **ROI:** Blake focuses on high-value activities, client satisfaction improves

### Phase 4: $75K-100K (Weeks 7-9)
**Trigger:** 15+ clients OR documentation backlog
**Third Hire:** Technical Writer (10-20 hrs/week)
- **Why:** Quality documentation = fewer support requests + better onboarding
- **When:** Support requests repeat OR new clients need custom guides
- **Cost:** $35-50/hr = $1,400-4,000/month
- **ROI:** Reduces support time 50%, improves client retention

## Role Priority Matrix

### Critical Path (hire in order):
1. **Python Developer** - Delivery capacity (revenue-blocking)
2. **VA/PM** - Operations efficiency (quality-blocking)
3. **Technical Writer** - Scalability (growth-blocking)

### Optional Roles (post-$100K):
- Designer/UI specialist (if building client portals)
- QA tester (if quality issues emerge)

## Where to Find Them

### Primary Platforms (ranked by quality):
1. **Upwork** - Largest pool, vetted profiles, escrow protection
2. **Toptal** - Pre-vetted top 3%, higher rates but faster hire
3. **LinkedIn** - Direct recruiting, no platform fees, relationship building
4. **We Work Remotely** - Remote-first talent, cultural fit
5. **Contra** - Commission-free, growing platform

### Search Strategies:
- **Upwork filters:** 90%+ job success, $earned >$10K, English fluency, availability
- **LinkedIn search:** "Python automation" + "freelance" OR "contractor" + location: remote
- **Keywords:** API integration, web scraping, RPA, workflow automation, Python scripting

## Interview Process (Fast Track - 48 Hour Hire)

### Stage 1: Application Screen (2 hours)
- Review portfolio/past work
- Check job success rate + reviews
- Verify hourly rate fits budget
- **Pass/fail:** Move 3-5 candidates to Stage 2

### Stage 2: Async Skills Test (24 hours)
**Python Dev:** Build a simple API scraper + format output
**VA/PM:** Organize a messy project brief into actionable tasks
**Writer:** Turn technical notes into user-friendly guide

**Scoring:**
- Quality of output (50%)
- Speed of completion (25%)
- Communication style (25%)

### Stage 3: Live Interview (30 min Zoom)
**Questions:**
1. Walk me through your approach to [skills test]
2. Describe a time you had to learn a new tool quickly
3. How do you handle unclear requirements?
4. Availability: Can you start this week? Hours/week?
5. Rate negotiation (if needed)

**Red flags:**
- Can't explain their own work
- Needs excessive hand-holding
- Poor communication clarity
- Unavailable for 2+ weeks

### Stage 4: Paid Trial (1 week, 5-10 hours)
- Small real project ($200-500 budget)
- Evaluate: quality, speed, communication, autonomy
- **Go/no-go decision:** Hire full-time OR part ways

## Contract Structure

### Key Terms:
- **Payment:** Weekly via Upwork/Wise (fast, builds trust)
- **Hours:** Track via Toggl or Upwork time tracker
- **Scope:** Flexible week-to-week based on client load
- **Termination:** Either party, 1 week notice
- **IP ownership:** ClawOps owns all work product
- **NDA:** Standard confidentiality (client data protection)

### Rate Negotiation:
- Start at mid-range of their posted rate
- Offer bonuses for exceptional work (10-20% project bonuses)
- Guarantee minimum hours/week if they commit availability

## Budget Planning

**Phase 2 ($25-50K revenue):**
- 1 Python dev (30 hrs/week): $6,000-7,200/month
- **Total:** ~$6,500/month (13% of revenue at $50K)

**Phase 3 ($50-75K revenue):**
- 1 Python dev (40 hrs/week): $8,000-9,600/month
- 1 VA/PM (30 hrs/week): $3,000-4,200/month
- **Total:** ~$12,000/month (16% of revenue at $75K)

**Phase 4 ($75-100K revenue):**
- 1 Python dev (40 hrs/week): $8,000-9,600/month
- 1 VA/PM (40 hrs/week): $4,000-5,600/month

### Profit Margin Protection:
- Keep contractor costs under 20% of revenue
- Gross margin target: 60-70% after contractor costs
- Blake's take-home + reinvestment: 50-60% of revenue

## Emergency Scaling Plan

**If we land a massive client ($50K+ contract):**
1. Hire 2 devs immediately (overlap for knowledge transfer)
2. Upgrade VA to full-time PM
3. Bring on QA specialist
4. Budget: 25-30% of contract value for delivery team

**If a contractor disappears:**
1. Activate backup from freelancer bench (see freelancer-bench.md)
2. Offer 20% rate premium for immediate start
3. Overlap new person with replacement for 1 week if possible

## Success Metrics

### Contractor Performance:
- Projects delivered on time (>90%)
- Client satisfaction scores (>4.5/5)
- Rework rate (<10%)
- Response time (<4 hours during work hours)

### Hiring Efficiency:
- Time to hire: <72 hours from posting
- Offer acceptance rate: >70%
- 90-day retention: >80%
- Cost per hire: <$500 (including trial projects)

## Key Principles

1. **Hire when it hurts, not when it's convenient** (revenue triggers prevent premature hiring)
2. **Trial before commitment** (1-week paid test eliminates bad fits)
3. **Optimize for speed** (48-hour hire cycle keeps momentum)
4. **Build a bench** (pipeline of pre-vetted talent for emergency scaling)
5. **Pay fairly, expect excellence** (mid-high rates = better talent = less management overhead)
