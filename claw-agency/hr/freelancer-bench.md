# ClawOps Freelancer Bench - Python Automation Specialists

## Purpose

This document serves as our pipeline of pre-vetted Python automation freelancers. When we land clients and need to scale fast, we pull from this bench instead of starting a cold search.

---

## Target Profile

**Core Requirements:**
- Python automation specialist (API integrations, web scraping, workflow automation)
- 90%+ job success rate (Upwork) or strong recommendations (LinkedIn)
- $30-60/hr rate range
- Available for contract work (20-40 hrs/week)
- English fluency (written communication is critical)

**Ideal Background:**
- 3-5 years professional Python experience
- Agency or freelance experience (used to moving fast)
- Portfolio of automation projects (scrapers, integrations, bots)
- Comfortable working with AI tools

---

## How to Use This List

### When You Need to Hire (Fast):

1. **Review the bench** (sorted by priority)
2. **Send outreach messages** to top 3-5 candidates (template below)
3. **Screen responses** within 24 hours
4. **Skills test** top 2-3 (paid, 2-3 hour task)
5. **Hire** best fit within 48 hours

### Maintaining the Bench:

- **Update quarterly:** Check if profiles are still active, rates current
- **Add new finds:** When you see great freelancers on projects, add them
- **Track outreach:** Note who you have contacted, when, and response

## Research Instructions (To Populate This List)

### Upwork Search Method

**Step 1: Search Setup**
- Go to Upwork.com > Find Talent
- Category: Web, Mobile & Software Dev > Scripting & Automation
- Search: "Python automation API integration"

**Step 2: Filter Settings**
- Job Success: 90% or higher
- Earned Amount: $10,000+ (proves track record)
- Hourly Rate: $30-75/hr (we filter to $30-60 manually)
- Location: Any (remote-friendly)
- Availability: Available Now or Available Soon

**Step 3: Profile Review**
- Check portfolio for relevant projects (scrapers, APIs, automation)
- Read reviews (look for "responsive," "quality," "fast")
- Verify rate fits budget
- Note timezone (prefer overlap with US hours, but not required)

**Step 4: Record Details**
- Name, Upwork profile link
- Rate, availability, timezone
- Specializations (what they are best at)
- Portfolio highlights (1-2 best projects)
- Notes (why they are a good fit)

### LinkedIn Search Method

**Step 1: Search Query**
- Search: "Python automation freelance"
- Filters: Open to freelance work, has recent activity
- Location: Remote or United States

**Step 2: Profile Review**
- Check experience section (Python, automation, APIs)
- Look for freelance/contractor roles (not just full-time)
- Check recommendations and endorsements
- Review posts/articles (are they knowledgeable?)

**Step 3: Record Details**
- Name, LinkedIn profile link
- Estimated rate (if listed or infer from experience)
- Current availability (active on LinkedIn = likely available)
- Specializations
- Notes
