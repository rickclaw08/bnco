# Quality Assurance Checklist — Client Deliverables
**Claw Agency — Standard Operating Procedure**
*Created: 2026-02-19*

---

## Before ANYTHING goes to a client, it passes through this checklist.

---

## 1. Content & Copy

- [ ] Spelling and grammar checked (run through tool + human eyes)
- [ ] Brand voice and tone match client guidelines
- [ ] All claims are accurate and sourced (if applicable)
- [ ] CTAs are clear and correct
- [ ] No placeholder or lorem ipsum text remaining
- [ ] Hashtags, tags, and mentions are correct
- [ ] Captions match the right visuals

## 2. Visual & Design

- [ ] Correct dimensions for target platform
- [ ] Brand colors, fonts, and logos used correctly
- [ ] Images are high resolution (not pixelated or stretched)
- [ ] No watermarks, stock photo artifacts, or uncleared assets
- [ ] Visual hierarchy is clear — key message is obvious
- [ ] Accessible contrast ratios (text legible over backgrounds)
- [ ] Consistent style across the batch/campaign

## 3. Technical

- [ ] All links work and go to correct destinations
- [ ] UTM parameters applied where needed
- [ ] Files exported in correct format (PNG, MP4, PDF, etc.)
- [ ] File names follow naming convention: `[client]-[type]-[date]-[version]`
- [ ] Responsive / mobile-friendly (if web deliverable)
- [ ] Load times acceptable (images optimized, videos compressed)

## 4. Strategy Alignment

- [ ] Deliverable matches the approved strategy/brief
- [ ] Content aligns with current campaign goals
- [ ] Posting schedule / timing is correct
- [ ] Target audience is appropriate
- [ ] Deliverable supports stated KPIs

## 5. Compliance & Legal

- [ ] No copyrighted material used without license
- [ ] Required disclaimers or disclosures included (ads, sponsorships)
- [ ] Privacy considerations met (no unauthorized personal info)
- [ ] Platform-specific policies followed (ad guidelines, content rules)

## 6. Final Review

- [ ] Reviewed by someone OTHER than the creator (fresh eyes)
- [ ] Client name spelled correctly everywhere
- [ ] Correct version number — no outdated drafts
- [ ] Deliverable matches what was promised in scope/timeline
- [ ] Packaged cleanly for client (organized folder, clear labels)
