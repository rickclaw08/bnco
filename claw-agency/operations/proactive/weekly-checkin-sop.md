# SOP: Weekly Client Check-Ins & Status Reports
**Claw Agency — Standard Operating Procedure**
*Created: 2026-02-19*

---

## Purpose

Maintain consistent communication with every active client. No surprises. No silence. Clients should always know what's happening.

---

## 1. Pre-Call Preparation (30 min before)

- [ ] Review task board — what's done, in progress, blocked
- [ ] Pull key metrics (engagement, ad spend, traffic, conversions — whatever applies)
- [ ] Note any blockers or items needing client input
- [ ] Review previous week's action items — are they closed?
- [ ] Prepare 2–3 talking points or recommendations

## 2. Check-In Agenda (15–30 min call or async update)

1. **Quick wins / completed work** — what we delivered this week
2. **In progress** — what's actively being worked on
3. **Upcoming** — what's next on the timeline
4. **Blockers / needs from client** — approvals, assets, decisions needed
5. **Metrics snapshot** — brief performance update (if applicable)
6. **Client questions / feedback** — open floor
7. **Action items** — clear next steps with owners and deadlines

## 3. Status Report Format

Send within 24 hours of check-in (or in lieu of call for async clients).

```
WEEKLY STATUS REPORT — [Client Name]
Week of: [Date]
Prepared by: [Name]

✅ COMPLETED THIS WEEK
- [Item 1]
- [Item 2]

🔄 IN PROGRESS
- [Item 1] — ETA: [date]
- [Item 2] — ETA: [date]

📋 UPCOMING (Next Week)
- [Item 1]
- [Item 2]

🚧 BLOCKERS / CLIENT ACTION NEEDED
- [Item — what we need, by when]

📊 KEY METRICS (if applicable)
- [Metric]: [Value] (↑/↓ vs last week)

📝 NOTES
- [Anything else relevant]
```

## 4. Delivery Method

| Tier | Format | Frequency |
|------|--------|-----------|
| Starter | Async message (email/Slack) | Weekly |
| Growth | Live call + written report | Weekly |
| Enterprise | Live call + written report + dashboard | Weekly |

## 5. Documentation

- Save all status reports in client's project folder
- Log action items in project management tool with due dates
- Flag any scope creep or timeline risks immediately to project lead

## 6. Escalation

If a client misses 2+ consecutive check-ins or is unresponsive:
1. Send a direct follow-up within 24 hours
2. Escalate to project lead / account manager on day 3
3. Formal outreach (call + email) by day 5
4. Document everything

---

> **Rule: No client goes more than 7 days without a touchpoint. Period.**
