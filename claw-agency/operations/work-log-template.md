# ClawOps Work Attribution Log

**Purpose:** Track who did what, when, and what it produced. Every task gets logged. No exceptions.

---

## How to Use

Copy the entry template below for each completed task. Append entries to:

```
claw-agency/operations/work-logs/YYYY-MM-DD.md
```

One file per day. All agents log to the same daily file.

---

## Entry Template

```markdown
### [Task Title]

| Field | Value |
|---|---|
| **Agent** | [Name / Role - e.g., Harper (COO)] |
| **Date** | [YYYY-MM-DD] |
| **Time Started** | [HH:MM EST] |
| **Time Completed** | [HH:MM EST] |
| **Duration** | [Xh Ym] |
| **Task Type** | [Operations / Client Work / Research / Internal / Learning] |
| **Priority** | [Critical / High / Medium / Low] |
| **Requested By** | [Name or "Self-initiated"] |

**Description:**
[1-3 sentences describing what was done.]

**Outcome:**
[What was the deliverable or result? Be specific. Link to files/artifacts if applicable.]

**Artifacts:**
- [Path to file, document, or deliverable produced]
- [Additional artifacts if any]

**Status:** [Completed / In Progress / Blocked / Handed Off]

**Notes:**
[Any context, blockers encountered, or follow-up needed. Leave blank if none.]

---
```
