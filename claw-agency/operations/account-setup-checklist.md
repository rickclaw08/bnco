# Account Setup Checklist

**Purpose:** Get Brand set up on every platform needed to start selling. Each account optimized, not just created.
**Owner:** Harper (COO) | Last updated: 2026-02-21

---

## 1. Upwork

**Why:** Highest volume of automation/integration project leads. Clients are pre-qualified (they have budget and a problem).

### Setup Steps

- [ ] Go to upwork.com/freelancers and click "Become a Freelancer"
- [ ] Use a professional email (brand@clawops.com or similar, not a personal Gmail)
- [ ] Complete profile with the following:

**Profile Title:**
```
Automation & Integration Specialist | Make, Zapier, n8n, API Development
```

**Overview (copy and customize):**
```
I build automation systems that save businesses 10-20+ hours per week.

What I do:
- Workflow automation (Make, Zapier, n8n)
- API integrations (connecting your tools so they talk to each other)
- CRM setup and automation (HubSpot, GoHighLevel, Salesforce)
- Data pipeline automation (spreadsheets, databases, reporting)
- Custom business process automation

How I work:
- Discovery call to understand your exact needs
- Clear scope and timeline before any work begins
- Weekly updates and transparent communication
- Full documentation and handoff so you own everything

Results I've delivered:
- [Add 2-3 specific results as you complete projects]

Let's talk about what's eating your team's time. I'll tell you if I can fix it.
```

**Additional profile setup:**
- [ ] Set hourly rate to $75-125/hr (positions as mid-high tier, filters out low-budget clients)
- [ ] Add skills: Automation, Zapier, Make (Integromat), n8n, API Integration, Python, JavaScript, CRM, Workflow Automation, Business Process Automation
- [ ] Set availability to "More than 30 hrs/week"
- [ ] Add portfolio items (even mockups or personal projects to start)
- [ ] Upload a professional headshot (not a logo, Upwork favors faces)
- [ ] Complete the skills tests for relevant categories (adds credibility)
- [ ] Set up payment method (for Connects purchases)
- [ ] Buy Connects: start with 60-80 ($0.15 each, so about $10-12)

**First week strategy:**
- [ ] Apply to 10-15 relevant jobs per day
- [ ] Write custom proposals for each (no copy-paste templates)
- [ ] Focus on jobs posted in the last 24 hours
- [ ] Target clients who have previously hired and paid on the platform
- [ ] Consider Upwork Plus ($14.99/mo) for better visibility

## 2. Fiverr

**Why:** Inbound lead generation. Clients come to you. Good for recurring small/medium projects and building reviews.

### Setup Steps

- [ ] Go to fiverr.com and sign up as a seller
- [ ] Complete profile with professional photo and bio

**Create these gigs (in order of priority):**

**Gig 1: Workflow Automation**
```
Title: I will build custom automation workflows using Make, Zapier, or n8n
Category: Programming & Tech > Other
Tags: automation, zapier, make, integromat, workflow, n8n, integration, API
```

**Gig 2: CRM Setup**
```
Title: I will set up and automate your CRM (HubSpot, GoHighLevel, Salesforce)
Category: Programming & Tech > Other
Tags: CRM, HubSpot, GoHighLevel, Salesforce, automation, setup
```

**Gig 3: API Integration**
```
Title: I will integrate your apps and tools with custom API connections
Category: Programming & Tech > Other
Tags: API, integration, webhook, REST API, automation, data sync
```

**For each gig:**
- [ ] Write a detailed description (at least 200 words)
- [ ] Add 3 pricing tiers (Basic: $150, Standard: $500, Premium: $1,500+)
- [ ] Set delivery times (Basic: 3 days, Standard: 7 days, Premium: 14 days)
- [ ] Add relevant FAQ (at least 3 questions)
- [ ] Upload a professional gig image (1280x769 px)
- [ ] Add a gig video if possible (increases conversion significantly)
- [ ] Set up requirements (what you need from the buyer before starting)

**Optimization:**
- [ ] Enable "Seller Plus" if available in your market
- [ ] Respond to all inquiries within 1 hour (Fiverr rewards fast response)
- [ ] After first 5 reviews, raise prices by 20-30%

## 3. LinkedIn (Business Presence)

**Why:** Professional credibility, networking, and inbound from decision-makers. Many enterprise and mid-market clients find vendors on LinkedIn.

### Setup Steps

**Personal Profile (Brand's LinkedIn):**
- [ ] Update headline:
```
Automation Consultant | Helping businesses save 10-20 hrs/week with smart workflows | Founder, ClawOps
```
- [ ] Update About section (similar to Upwork overview but more personal, story-driven)
- [ ] Add "ClawOps" as current position under Experience
- [ ] Add relevant skills and get endorsements
- [ ] Upload a professional headshot and banner image
- [ ] Set profile to "Open to Work" for consulting/freelance (visible to recruiters only, not public)

**Company Page (ClawOps):**
- [ ] Go to linkedin.com/company/setup/new/
- [ ] Company name: ClawOps
- [ ] Industry: Information Technology & Services
- [ ] Company size: 2-10
- [ ] Type: Privately Held
- [ ] Upload logo and banner image
- [ ] Write the company description:
```
ClawOps builds automation systems that save businesses time and money. We design, build, and maintain custom workflows that connect your tools, eliminate manual work, and let your team focus on what matters.

Services: Workflow Automation, API Integration, CRM Setup, Business Process Automation, Custom Integrations

Tools: Make (Integromat), Zapier, n8n, Python, JavaScript, REST APIs
```
- [ ] Add a website URL (even if it's a simple landing page)
- [ ] Post 2-3 initial posts (case studies, tips, or insights about automation)

**Outreach strategy:**
- [ ] Connect with 20-30 people per day (ops managers, CTOs, founders in target industries)
- [ ] Engage on 5-10 relevant posts daily (comment with genuine value, not spam)
- [ ] Post 3-5 times per week (automation tips, client wins, behind-the-scenes)
- [ ] DM warm leads with value-first messages (not pitches)

## 4. Stripe

**Why:** Payment processing. Accept credit cards, send invoices, handle subscriptions for retainers.

### Setup Steps

- [ ] Go to stripe.com and create an account
- [ ] Use business email (brand@clawops.com or similar)
- [ ] Select account type: Individual/Sole Proprietor (or LLC if formed)
- [ ] Complete identity verification (government ID, SSN/EIN)
- [ ] Add bank account for payouts
- [ ] Set payout schedule to "Daily" or "Weekly" (daily recommended for cash flow)

**Configure for client payments:**
- [ ] Set up Stripe Invoicing (stripe.com/invoicing)
  - Create invoice templates with ClawOps branding
  - Set default payment terms (Net-15 recommended)
  - Enable automatic payment reminders (3 days before due, day of, 3 days after)
- [ ] Create Payment Links for common packages:
  - [ ] "Automation Consultation - $0" (free discovery call booking, for tracking)
  - [ ] "Small Automation Project - $1,500" (or your starting price)
  - [ ] "Custom Project - Variable" (use invoicing for these)
- [ ] Set up Stripe Tax if selling to clients in states with sales tax obligations
- [ ] Enable Stripe Radar for fraud protection (enabled by default)

**Subscription setup (for retainers):**
- [ ] Create products in Stripe:
  - [ ] "Monthly Retainer - Basic" - $2,000/mo
  - [ ] "Monthly Retainer - Standard" - $3,500/mo
  - [ ] "Monthly Retainer - Premium" - $5,000/mo
- [ ] Set up subscription billing with automatic collection

**Accounting integration:**
- [ ] Connect Stripe to QuickBooks, Xero, or Wave (for bookkeeping)
- [ ] Or at minimum, export monthly transaction reports

## 5. Additional Platforms (Recommended)

### Calendly or Cal.com (Booking)
- [ ] Create a free account
- [ ] Set up a "Discovery Call" event type (30 min)
- [ ] Set up a "Kickoff Call" event type (45 min)
- [ ] Connect to Google Calendar
- [ ] Add the booking link to all profiles (Upwork, Fiverr, LinkedIn, website)

### Google Workspace (Professional Email)
- [ ] Set up brand@clawops.com (or similar)
- [ ] Configure email signature with name, title, booking link
- [ ] Set up Google Drive folder structure:
  - Clients/
  - Templates/
  - Internal/
  - Sales/

### Simple Website/Landing Page
- [ ] Create a one-page site on Carrd ($19/yr), Framer, or similar
- [ ] Include: what you do, who you help, 2-3 results, booking link, contact
- [ ] Add to all platform profiles

---

## Priority Order

Do these in order. Each one builds on the previous:

1. **Stripe** (you need to accept payments before anything else)
2. **Google Workspace** (professional email for all other signups)
3. **Calendly/Cal.com** (booking link for all profiles)
4. **LinkedIn** (credibility and networking, start immediately)
5. **Upwork** (highest volume lead source, apply aggressively)
6. **Fiverr** (set up gigs, then let inbound come to you)
7. **Website** (nice to have, do this in Week 2)
