# Client Onboarding Checklist

## Overview
This checklist covers the complete onboarding workflow from contract signature to project kickoff. Goal: Get clients set up, aligned, and excited within 7 days.

---

## Phase 1: Contract to Welcome (Day 0-1)

### Immediate Actions (Within 24 Hours of Signed Contract)
- [ ] Send welcome email (see communication-templates.md)
- [ ] Create client folder in project management system
- [ ] Set up dedicated Slack channel or communication hub
- [ ] Add client to project tracking system
- [ ] Schedule kickoff call (aim for within 3-5 business days)
- [ ] Send calendar invite with kickoff agenda
- [ ] Assign project lead and technical lead

### Client Information Gathering
- [ ] Collect primary contact details (name, email, phone, timezone)
- [ ] Identify key stakeholders and decision makers
- [ ] Understand preferred communication channels
- [ ] Confirm billing contact if different from primary
- [ ] Get access to any required accounts or systems
- [ ] Collect brand assets (logo, colors, style guide if applicable)

---

## Phase 2: Discovery & Alignment (Day 2-5)

### Pre-Kickoff Preparation
- [ ] Review signed proposal and scope document
- [ ] Research client's industry and competitors
- [ ] Prepare kickoff presentation deck
- [ ] Draft initial project timeline
- [ ] Prepare technical questions for discovery
- [ ] Review any materials client provided during sales

### Kickoff Call Agenda (60-90 minutes)
- [ ] Introductions and team roles (10 min)
- [ ] Review project scope and objectives (15 min)
- [ ] Walk through timeline and milestones (10 min)
- [ ] Technical discovery session (30 min)
  - Current tools and systems
  - Pain points and bottlenecks
  - Integration requirements
  - Data access and permissions
  - Security and compliance needs
- [ ] Communication protocol overview (5 min)
- [ ] Next steps and expectations (10 min)
- [ ] Q&A (10 min)

### Post-Kickoff Actions (Same Day)
- [ ] Send kickoff meeting recap email
- [ ] Document all technical requirements
- [ ] Create detailed project plan with dates
- [ ] Set up recurring check-in meetings
- [ ] Request any additional access needed
- [ ] Share project tracker link with client

---

## Phase 3: Technical Setup (Day 3-7)

### Access & Permissions
- [ ] Collect all necessary API keys
- [ ] Set up development/staging environments
- [ ] Configure access to client systems
- [ ] Set up version control repository
- [ ] Document all credentials in secure vault
- [ ] Test all integrations and connections

### Project Infrastructure
- [ ] Create project documentation hub
- [ ] Set up task tracking board
- [ ] Initialize technical architecture document
- [ ] Create testing environment
- [ ] Set up monitoring/logging (if applicable)
- [ ] Configure backup procedures

### Client Portal Setup
- [ ] Grant access to communication channel
- [ ] Share project dashboard link
- [ ] Provide documentation access
- [ ] Set up feedback submission process
- [ ] Share support contact information

---

## Phase 4: Kickoff Complete (Day 7)

### Final Checks
- [ ] All stakeholders have access to communication channels
- [ ] Project plan approved by client
- [ ] Technical requirements documented
- [ ] Development environment ready
- [ ] First sprint/milestone defined
- [ ] Weekly check-in scheduled
- [ ] Emergency escalation path established

### Send Onboarding Complete Email
- [ ] Thank client for smooth onboarding
- [ ] Confirm next milestone and date
- [ ] Reiterate how to reach team
- [ ] Set expectations for first update

---

## Onboarding Success Criteria

**Client should have:**
- Clear understanding of project timeline
- Access to all communication channels
- Direct contact with project team
- Visibility into project progress
- Confidence in our process

**Team should have:**
- Complete technical requirements
- Access to all necessary systems
- Documented scope and deliverables
- Clear success criteria
- Risk mitigation plan

---

## Tier-Specific Notes

### Starter ($500 one-time)
- Streamlined onboarding (2-3 days)
- Single kickoff call (30-45 min)
- Email as primary communication
- Monthly check-ins only

### Growth ($2K/month)
- Full 7-day onboarding
- Weekly check-ins
- Dedicated Slack channel
- Bi-weekly strategy calls

### Enterprise ($5-15K/month)
- White-glove onboarding (custom timeline)
- Daily/weekly standups as needed
- Dedicated account manager
- Custom SLAs and reporting
- Quarterly business reviews

---

## Red Flags to Watch For

- Client unresponsive during onboarding (delays kill momentum)
- Scope creep starting before project begins
- Unclear decision-making authority
- Technical access issues or resistance
- Misaligned expectations on timeline

**Action:** Address immediately with project lead and potentially escalate to account owner.
