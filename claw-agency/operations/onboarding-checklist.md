# Client Onboarding Checklist

Use this for **every new client**. Copy to their Notion/Trello project and check off as you go.

---

## Phase 1: Pre-Contract
- [ ] Intake form received and reviewed
- [ ] Lead scored (Hot / Warm / Cold)
- [ ] Discovery call scheduled
- [ ] Discovery call completed
- [ ] Proposal drafted and sent
- [ ] Proposal accepted / negotiation complete
- [ ] Contract sent (DocuSign/PandaDoc)
- [ ] Contract signed by both parties
- [ ] 50% deposit invoiced (Stripe)
- [ ] Deposit received

## Phase 2: Workspace Setup
- [ ] Trello board / Notion project created from template
- [ ] GitHub repo created (private, branch protection on `main`)
- [ ] Client added to Slack/Discord channel
- [ ] Client credentials/API keys collected securely (1Password or secure form)
- [ ] Environment variables documented (`.env.example`)
- [ ] Deployment environment provisioned (Vercel/Railway)
- [ ] Project folder structure initialized

## Phase 3: Kickoff
- [ ] Kickoff call scheduled
- [ ] Kickoff agenda sent to client (24h before)
- [ ] Kickoff call completed
- [ ] Milestones and timeline confirmed in writing
- [ ] Point of contact and communication norms confirmed
- [ ] First milestone check-in date set
- [ ] First status update sent

## Phase 4: Ongoing (per milestone)
- [ ] Milestone deliverables built
- [ ] Internal QA completed (see qa-process.md)
- [ ] Demo/review call with client
- [ ] Client feedback incorporated
- [ ] Milestone signed off by client
- [ ] Payment tranche invoiced (if milestone-based)

## Phase 5: Handoff & Close
- [ ] Final QA pass completed
- [ ] Client UAT period (3 business days) — feedback collected
- [ ] UAT fixes applied
- [ ] Handoff package delivered:
  - [ ] Technical documentation
  - [ ] User guide / SOP
  - [ ] Admin guide
  - [ ] Loom walkthrough recorded
- [ ] Repo access transferred / granted
- [ ] Knowledge transfer call completed
- [ ] Final invoice sent (remaining balance)
- [ ] Final payment received
- [ ] 14-day support period starts (note end date: _______)
- [ ] Maintenance retainer offered
- [ ] Client testimonial / review requested
- [ ] Project archived (board → Done, repo tagged)
