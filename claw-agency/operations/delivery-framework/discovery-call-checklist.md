# Discovery Call Checklist

**Use this during every discovery/sales call. Fill it out live or immediately after.**

---

## Pre-Call Prep (5 min before)

- [ ] Research the prospect's company (website, LinkedIn, social)
- [ ] Check if they were referred and by whom
- [ ] Review any pre-call questionnaire responses
- [ ] Have the pricing reference open but not visible

---

## Call Structure (30-60 min)

### 1. Rapport and Context (5 min)

- [ ] Introduce yourself and ClawOps (30 seconds, not a pitch)
- [ ] Ask: "What made you reach out?" or "What prompted this call?"
- [ ] Understand their role and decision-making authority

**Notes:**
```
Role: 
Company: 
How they found us: 
Decision maker? Yes / No - who else is involved? 
```

### 2. Current State (10 min)

- [ ] Ask: "Walk me through how [process] works today, step by step."
- [ ] Ask: "How much time does this take per week/month?"
- [ ] Ask: "Who's involved in this process?"
- [ ] Ask: "What tools are you currently using?"
- [ ] Ask: "What's the most painful or error-prone part?"

**Notes:**
```
Current process: 
Time spent: 
People involved: 
Tools in use: 
Biggest pain point: 
```

### 3. Desired State (10 min)

- [ ] Ask: "In a perfect world, how would this work?"
- [ ] Ask: "What would success look like 30 days after we finish?"
- [ ] Ask: "Are there specific metrics you're trying to improve?"
- [ ] Ask: "Have you tried to solve this before? What happened?"

**Notes:**
```
Ideal outcome: 
Success metrics: 
Previous attempts: 
```

### 4. Scope and Feasibility (10 min)

- [ ] Identify the core automation(s) needed
- [ ] Ask: "Are there integrations required?" (CRM, email, payment, etc.)
- [ ] Ask: "How many users/records/transactions are we talking about?"
- [ ] Ask: "Are there compliance or security requirements?"
- [ ] Mentally map to a project type (simple, multi-step, full system)

**Notes:**
```
Core automations: 
Integrations needed: 
Volume/scale: 
Compliance/security: 
Estimated project type: Simple / Multi-step / Full System / Enterprise
```

### 5. Timeline and Budget (5 min)

- [ ] Ask: "When do you need this by?"
- [ ] Ask: "Have you budgeted for this project?"
- [ ] If they won't share budget: "Our projects typically range from $1,500 to $15,000+ depending on scope. Does that range work for you?"
- [ ] Qualify: Is this a real project or are they just exploring?

**Notes:**
```
Timeline: 
Budget range: 
Readiness: Ready to start / Planning / Just exploring
```

### 6. Next Steps (5 min)

- [ ] Summarize what you heard (confirm understanding)
- [ ] Explain next steps: "I'll put together a proposal with scope, timeline, and pricing. You'll have it within 48 hours."
- [ ] Confirm the best email for the proposal
- [ ] Ask: "Is there anyone else who needs to review the proposal?"
- [ ] Set a follow-up date: "Can we schedule a quick call on [day] to walk through it?"

**Notes:**
```
Proposal email: 
Other stakeholders: 
Follow-up date: 
```

---

## Post-Call Actions (Within 1 Hour)

- [ ] Send a thank-you email with recap of key points
- [ ] Log the prospect in the CRM/tracker
- [ ] Draft the proposal (use the requirements template for scoping)
- [ ] Set a calendar reminder for the follow-up date
- [ ] If not a fit: send a polite decline email within 24 hours
