## USAGE RULES

### Daily Updates (Each Agent)
1. Move items between columns as status changes
2. Update "Status: X% complete" and task checkboxes
3. Add blockers immediately when identified
4. Update "Next Steps" so anyone can understand where things stand

### Harper's Daily Review (EOD)
1. Scan for new blockers (resolve or escalate within 24 hours)
2. Check items stuck in "In Progress" > 3 days (intervene or replan)
3. Review "Review" column (chase approvals if > 2 days)
4. Archive "Done" items on Friday EOD

### Weekly Cleanup (Friday EOD)
1. Archive all "Done" items to `/operations/project-archives/YYYY-MM-DD-done.md`
2. Re-prioritize "Backlog" based on weekly planning
3. Assess blocked items (escalate or deprioritize)
4. Update board health metrics

### Monthly Deep-Dive (Last Friday)
1. Review average cycle time (Backlog to Done)
2. Identify bottlenecks (what's always stuck?)
3. Assess backlog size (growing or shrinking?)
4. Adjust process if needed
