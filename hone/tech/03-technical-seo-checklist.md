# HONE — Technical SEO Checklist

> Shopify-specific SEO for honegrooming.com

---

## 1. Crawlability & Indexing

- [ ] Submit XML sitemap to Google Search Console (`/sitemap.xml` — auto-generated by Shopify)
- [ ] Submit sitemap to Bing Webmaster Tools
- [ ] Verify `robots.txt` is correct (Shopify auto-generates; review at `/robots.txt`)
- [ ] Ensure no important pages are blocked (check via GSC → Coverage)
- [ ] Remove store password before launch (Settings → Preferences → disable password)
- [ ] Canonical tags: Shopify auto-adds; verify they're correct on product pages with multiple collections

## 2. Site Structure & URLs

- [ ] Keep URLs short and keyword-rich:
  - Products: `/products/beard-oil-smokewood`
  - Collections: `/collections/beard-oils`
  - Pages: `/pages/about`
  - Blog: `/blogs/hone-journal/post-title`
- [ ] Avoid changing URL handles after launch (causes 301 redirect chains)
- [ ] Set up **URL redirects** for any old/changed URLs (`Online Store → Navigation → URL Redirects`)
- [ ] Shopify limitation: `/collections/` and `/products/` prefixes are fixed — work with them

## 3. On-Page SEO — Every Page

- [ ] **Title tag**: unique, <60 chars, format: `Primary Keyword | HONE Grooming`
- [ ] **Meta description**: unique, 150-160 chars, includes CTA
- [ ] **H1**: one per page, matches page intent
- [ ] **Header hierarchy**: H1 → H2 → H3 (no skipping levels)
- [ ] **Image alt text**: descriptive, includes keywords naturally (every image)
- [ ] **Image file names**: descriptive before upload (`hone-beard-oil-smokewood-2oz.jpg`)
- [ ] **Internal links**: link between related products, collections, and blog posts

## 4. Product Page SEO

- [ ] Unique product descriptions (NO manufacturer copy-paste)
- [ ] SEO title format: `[Product Name] - [Key Benefit] | HONE Grooming`
- [ ] Meta description: benefit-driven, 150-160 chars
- [ ] Product structured data: Shopify auto-adds `Product` schema — verify with [Rich Results Test](https://search.google.com/test/rich-results)
- [ ] Review structured data: Judge.me adds `AggregateRating` — verify stars appear in SERPs
- [ ] Breadcrumbs: ensure theme includes breadcrumb navigation (helps Google understand hierarchy)

## 5. Collection Page SEO

- [ ] Add **unique collection descriptions** (100-200 words, keyword-optimized)
- [ ] SEO title: `[Collection Name] - Men's Beard Care | HONE Grooming`
- [ ] Collection descriptions should appear above OR below the product grid
- [ ] Avoid thin content: each collection needs real descriptive text

## 6. Performance & Core Web Vitals

- [ ] Target **90+ Lighthouse score** on mobile
- [ ] **Images**: compress all images <500KB; use WebP where possible
- [ ] **Lazy loading**: enable for below-fold images (Dawn does this by default)
- [ ] **Minimize apps**: each app adds JS; only install what you need
- [ ] **Theme code**: avoid excessive custom Liquid/JS
- [ ] **Fonts**: limit to 2 font families; use `font-display: swap`
- [ ] Test with:
  - [PageSpeed Insights](https://pagespeed.web.dev/)
  - [GTmetrix](https://gtmetrix.com/)
  - Chrome DevTools Lighthouse
- [ ] **Core Web Vitals targets**:
  - LCP (Largest Contentful Paint): <2.5s
  - FID/INP (Interaction to Next Paint): <200ms
  - CLS (Cumulative Layout Shift): <0.1

## 7. Mobile Optimization

- [ ] Test all pages on mobile devices (real devices, not just DevTools)
- [ ] Tap targets: buttons/links minimum 48×48px
- [ ] Font size: minimum 16px body text on mobile
- [ ] No horizontal scrolling
- [ ] Mobile navigation works smoothly
- [ ] Cart drawer functions properly on mobile
- [ ] Checkout is frictionless on mobile

## 8. Structured Data / Schema

Shopify + theme + Judge.me should auto-generate these — **verify all**:

- [ ] `Organization` schema on homepage (add via theme code if missing)
- [ ] `Product` schema on product pages (price, availability, reviews)
- [ ] `AggregateRating` on product pages (via Judge.me)
- [ ] `BreadcrumbList` on all pages
- [ ] `Article` schema on blog posts
- [ ] `WebSite` schema with `SearchAction` (sitelinks search box)
- [ ] Validate all at [schema.org validator](https://validator.schema.org/) or Google Rich Results Test

## 9. Content & Blog SEO

- [ ] Publish **3-5 blog posts before launch** (shows Google the site has depth)
- [ ] Target long-tail keywords:
  - "best beard oil for dry skin"
  - "how to apply beard balm"
  - "beard grooming routine for beginners"
  - "beard oil vs beard balm"
  - "natural beard care products"
- [ ] Each post: 800-1500 words, includes internal links to products
- [ ] Add **FAQ schema** to relevant blog posts and product pages
- [ ] Blog post URL format: `/blogs/hone-journal/[keyword-rich-handle]`

## 10. Local & Brand SEO

- [ ] Create **Google Business Profile** (even for e-commerce — establishes brand entity)
- [ ] Consistent **NAP** (name, address, phone) across all platforms
- [ ] Set up social profiles and link from website:
  - Instagram, Facebook, TikTok, YouTube, X/Twitter
- [ ] Claim brand on major directories

## 11. Security & Trust

- [ ] SSL certificate: auto-provided by Shopify — verify green lock
- [ ] Force HTTPS: `Settings → Domains → SSL → Enable`
- [ ] HSTS: enabled by default on Shopify

## 12. International / hreflang (Future)

- [ ] If adding international markets, add `hreflang` tags
- [ ] Shopify Markets handles this automatically when configured
- [ ] Not needed at launch (US-only)

## 13. Post-Launch Monitoring

- [ ] **Google Search Console**: monitor weekly
  - Index coverage, crawl errors, Core Web Vitals, search performance
- [ ] **GA4**: monitor organic traffic, landing pages, conversion rate
- [ ] Set up **GSC email alerts** for critical issues
- [ ] Monthly: check for broken links (use Screaming Frog or Ahrefs free)
- [ ] Monthly: review and update meta titles/descriptions for underperforming pages
- [ ] Track keyword rankings for target terms (use Ahrefs, SEMrush, or free alternatives)
