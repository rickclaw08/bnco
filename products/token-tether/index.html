Up and Running in 3 Steps

No complicated setup. No code changes. Just run the installer and point your clients to the new port.

Install

Unzip the download, run ./install.sh. It checks your system, installs dependencies, and walks you through configuration interactively. Docker also supported.

Configure your limits

Set your hard limit (e.g. $50/day), per-session cap, warning threshold, and alert channels. All via a single .env file.

Redirect clients

Point your OpenClaw clients to port 18788 instead of 18789. Done. TokenTether proxies everything transparently and enforces your limits.

.env.example 
# ---- Network ----
TT_PROXY_PORT=18788
TT_GATEWAY_HOST=127.0.0.1
TT_GATEWAY_PORT=18789

# ---- Spending Limits ----
TT_HARD_LIMIT_USD=50.00
TT_SESSION_LIMIT_USD=10.00
TT_WARN_THRESHOLD_PCT=0.80

# ---- Reset ----
TT_RESET_INTERVAL=24h

# ---- Alerts (Slack, Discord, or any webhook) ----
TT_WEBHOOK_URL=https://hooks.slack.com/services/YOUR/WEBHOOK

# ---- Dashboard ----
TT_DASHBOARD_ENABLED=true
TT_DASHBOARD_PORT=18790

# ---- Behavior ----
TT_LIMIT_ACTION=block
TT_PASSTHROUGH=false

Frequently Asked Questions

Does TokenTether modify my API requests or responses?
No. TokenTether is a transparent proxy. It reads messages to extract usage data, but it never modifies, delays, or alters any traffic. Your clients and the gateway see identical data with or without TokenTether in the middle.
What happens if TokenTether crashes?
Your OpenClaw gateway keeps running normally. TokenTether is a safety layer, not a dependency. If it goes down, you lose the spending protection until it restarts, but nothing else breaks. State is persisted to disk and recovered on restart.
Which models does it support?
All of them. TokenTether ships with built-in pricing for 20+ models across OpenAI, Anthropic, Google, and DeepSeek. Unknown models use conservative default pricing, and you can add custom pricing via the MODEL_PRICING_JSON environment variable.
Can I run it on my existing server alongside OpenClaw?
Yes. TokenTether requires Node.js 18+ and minimal resources (under 64MB RAM, negligible CPU). Run it directly with Node.js or via Docker. It is designed to co-exist on the same machine as your OpenClaw gateway.
Is this a subscription?
No. $49 one-time purchase. You get the full source code, run it on as many of your own machines as you want, and receive free updates for all 1.x releases. No recurring fees.
Can I test it before going live?
Yes. Set TT_PASSTHROUGH=true in your .env file. TokenTether will monitor and log everything but never block or kill sessions. Once you are comfortable with the data, flip it to false to enable enforcement.
Do I get support?
