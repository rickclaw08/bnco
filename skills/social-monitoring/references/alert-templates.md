# Alert Templates

Pre-written alert messages for common social monitoring scenarios. Copy and customize.

## Hot Lead Alerts

### Form Submission
```
New lead from website form:
- Name: [name]
- Email: [email]
- Company: [company]
- Need: [what they asked for]
- Estimated tier: [Starter/Growth/Sprint/Enterprise]
- Action: Respond within 2 hours
```

### Reddit DM Lead
```
Reddit DM from potential client:
- User: u/[username]
- Context: [what they said]
- Subreddit origin: r/[sub]
- Interest level: [high/medium]
- Recommended response: [brief suggestion]
```

### Direct Email Lead
```
Direct email lead:
- From: [email]
- Subject: [subject]
- Summary: [1-2 sentences]
- Budget signals: [any mentioned]
- Next step: [respond/schedule call/send pricing]
```

## Engagement Alerts

### Viral Comment
```
Reddit comment getting traction:
- Subreddit: r/[sub]
- Comment: "[first 50 chars...]"
- Upvotes: [count]
- Replies: [count]
- Opportunity: [any follow-up potential]
```

### Negative Engagement
```
Heads up - negative response on Reddit:
- Subreddit: r/[sub]
- Issue: [what happened]
- Downvotes/criticism: [details]
- Recommended action: [respond/ignore/delete]
```

## Status Updates

### Daily Social Summary
```
Social check complete:
- Reddit: [X] replies, karma [+/-X]
- Discord: [X] relevant messages
- Email: [X] new submissions
- Hot leads: [count]
- Top priority: [most urgent item]
```

### Weekly Social Report
```
Weekly social metrics:
- Reddit karma: [total] ([+/-X] this week)
- Comments posted: [X]
- Leads generated: [X]
- Leads in pipeline: [X]
- Conversion: [X]%
- Top performing subreddit: r/[sub]
```
