## Keyword Research Process

When researching new keywords:
1. Start with the core topic
2. Use web_search to find related searches
3. Check competitor content for keyword gaps
4. Look for questions people ask (Reddit, Quora, forums)
5. Prioritize low difficulty, medium volume keywords
6. Map each keyword to a specific content piece
7. Add to this file for tracking
