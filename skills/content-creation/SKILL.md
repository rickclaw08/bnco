## Workflow

### Step 1: Define the Content Brief

1. Identify the content type: blog, social post, newsletter, or marketing copy
2. Determine the target audience segment
3. Select 1-3 primary keywords from references/seo-keywords.md
4. Define the goal: awareness, engagement, conversion, or retention
5. Set word count target:
   - Blog post: 800-1,500 words
   - Social post: 50-280 characters (platform dependent)
   - Newsletter: 300-600 words
   - Marketing copy: varies by placement

### Step 2: SEO Keyword Research (Blog Posts)

1. Check references/seo-keywords.md for existing target keywords
2. If new topic, research keywords:
   - Use web_search to find what people are searching for
   - Look for long-tail variations with lower competition
   - Check competitor content for keyword gaps
3. Select primary keyword and 2-3 secondary keywords
4. Map keywords to content sections naturally (no keyword stuffing)

### Step 3: Content Structure

**Blog Post Structure:**
```
Title (include primary keyword, under 60 chars)
Meta description (include primary keyword, 150-160 chars)
Introduction (hook + problem statement, 100-150 words)
H2: [Section with secondary keyword]
  - Supporting points
  - Examples or data
H2: [Section with secondary keyword]
  - Supporting points
  - Examples or data
H2: [Practical takeaway or how-to]
  - Actionable steps
Conclusion (CTA, 50-100 words)
```

**Social Post Structure:**
```
Hook (first line grabs attention)
Value (insight, tip, or story)
CTA (question, link, or next step)
Hashtags (if platform supports them)
```

**Newsletter Structure:**
```
Subject line (under 50 chars, curiosity or value-driven)
Preview text (complements subject, under 90 chars)
Greeting
Lead story (biggest value item)
Secondary items (2-3 shorter pieces)
CTA
Sign-off
```

### Step 4: Write First Draft

1. Load references/brand-voice.md and internalize the voice
2. Write the draft following the chosen structure
3. Rules while writing:
   - NEVER use em dashes. Use commas, periods, colons, or parentheses instead
   - Keep sentences under 25 words when possible
   - Use active voice
   - Be direct and pragmatic, not flowery
   - Include specific numbers and examples over vague claims
   - Write at an 8th-grade reading level
   - Break up long paragraphs (max 3-4 sentences)

### Step 5: Brand Voice Check

1. Run through the brand voice checklist:
   - [ ] No em dashes anywhere in the content
   - [ ] Tone is pragmatic and direct
   - [ ] No corporate jargon or buzzwords
   - [ ] Sounds like a smart friend, not a press release
   - [ ] ClawOps value props are accurate
   - [ ] No exaggerated claims
2. If any check fails, revise before proceeding

### Step 6: Platform Formatting

1. Check references/platform-formatting.md for the target platform
2. Apply platform-specific rules:
   - **Discord:** No markdown tables. Use bullet lists. Keep under 2000 chars per message
   - **WhatsApp:** No headers (# syntax). Use **bold** or CAPS for emphasis
   - **Blog/Web:** Full markdown supported
   - **Twitter/X:** 280 char limit. No markdown
   - **Reddit:** Markdown supported. Match subreddit tone
   - **Email/Newsletter:** HTML formatting. Keep images small

### Step 7: Review and Polish

1. Read the content aloud (mentally) for flow
2. Cut anything that does not add value
3. Verify all links and references are accurate
4. Check spelling and grammar
5. Ensure CTA is clear and compelling

### Step 8: Publish or Deliver

1. For blog posts: Format for the target CMS
2. For social posts: Use the appropriate platform tool or browser
3. For newsletters: Format for the email platform
4. Log the published content in today's memory file
5. Update references/content-calendar-template.md with the published piece

## Content Calendar Integration

When creating content, check references/content-calendar-template.md to:
- Avoid duplicate topics
- Maintain consistent posting cadence
- Align with planned campaigns or launches
- Track what has been published and what is upcoming

## Quality Standards

- Every piece must pass the brand voice check (Step 5)
- Blog posts must include at least one primary keyword in title, H1, and first paragraph
- Social posts must have a clear hook in the first line
- Newsletters must have a compelling subject line
- All content must include a call to action
