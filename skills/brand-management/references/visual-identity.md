## Typography

### Font Stack
- **Headings:** System font stack (Inter preferred, falls back to -apple-system, BlinkMacSystemFont, Segoe UI, Roboto)
- **Body:** Same system font stack
- **Code:** Fira Code, JetBrains Mono, or system monospace

### Type Scale
- H1: 2.25rem (36px), bold
- H2: 1.875rem (30px), semibold
- H3: 1.5rem (24px), semibold
- H4: 1.25rem (20px), medium
- Body: 1rem (16px), regular
- Small: 0.875rem (14px), regular
- Caption: 0.75rem (12px), regular

### Type Rules
- Line height: 1.5 for body, 1.2 for headings
- Maximum line width: 65-75 characters for readability
- Use sentence case for headings (not Title Case except for proper nouns)

## Logo Usage

### Rules
- Minimum clear space: equal to the height of the "C" in ClawOps on all sides
- Minimum size: 24px height for digital, 0.5 inch for print
- Do not stretch, rotate, or distort
- Do not change the colors outside approved palette
- Do not add effects (shadows, gradients, outlines)

### Approved Variations
1. Full color on dark background (green accent on navy)
2. Full color on light background (navy on white)
3. Single color white (for dark or busy backgrounds)
4. Single color dark (for light backgrounds)

## Imagery

### Style
- Clean, minimal, and modern
- Dark backgrounds with green accents preferred
- Screenshots should be clean (no cluttered desktops)
- Diagrams should use the brand color palette

### Do Not Use
- Generic stock photos (handshakes, pointing at screens, "diverse team laughing")
- Overly futuristic robot imagery
- Clip art or low-quality graphics
- Images with text that is not in our brand fonts

### Screenshots
- Use dark mode when available
- Crop to show only relevant content
- Add subtle borders or shadows if needed for definition
- Annotate with green (#4ade80) highlights or arrows

## Layout Principles

1. **Generous whitespace.** Let content breathe
2. **Clear hierarchy.** One primary action per view
3. **Consistent spacing.** Use an 8px grid system
4. **Mobile-first.** Design for small screens, scale up
5. **Accessibility.** Minimum 4.5:1 contrast ratio for text
