## Workflow

### Step 1: Identify Outreach Opportunity

1. Determine the outreach type:
   - **Reactive:** Respond to an existing thread, question, or discussion
   - **Proactive:** Create a new post sharing value, insight, or a resource
2. Choose the platform: Reddit, Discord, or other forum
3. Check references/ for platform-specific strategy

### Step 2: Find Relevant Threads (Reactive Outreach)

**Reddit:**
1. Browse target subreddits from references/reddit-strategy.md
2. Search for threads matching ClawOps topics:
   - AI agents, automation, agentic workflows
   - Small business operations, consulting tools
   - Developer tools, open source AI
3. Filter for threads that are:
   - Less than 24 hours old (for visibility)
   - Asking a genuine question ClawOps can help with
   - Not already saturated with responses
4. Avoid threads where a response would feel forced or spammy

**Discord:**
1. Check target Discord servers from references/discord-strategy.md
2. Look for help channels, general discussion, or show-and-tell channels
3. Find conversations where ClawOps experience is genuinely relevant

### Step 3: Craft the Response

1. Load references/comment-templates.md for structure inspiration
2. Write the response following these rules:
   - **Lead with value.** Answer the question or add insight first
   - **Be genuine.** Write as a practitioner, not a marketer
   - **Mention ClawOps only if directly relevant.** Never force it
   - **Share experience.** "We built X and learned Y" is better than "Try our product"
   - **No em dashes.** Use commas, periods, or parentheses
   - **Match the community tone.** Reddit is casual. Discord varies by server
   - **Keep it concise.** 2-4 paragraphs max for comments. Shorter is better
3. Include a subtle CTA only when natural:
   - Link to a relevant blog post or resource
   - Mention the project name if contextually appropriate
   - Offer to share more details if someone is interested

### Step 4: Review Before Posting

1. Self-promotion check:
   - [ ] Would this comment be valuable even without any ClawOps mention?
   - [ ] Does it answer the question or add to the discussion?
   - [ ] Would a regular community member write something like this?
   - [ ] Is the tone appropriate for this specific community?
2. Brand voice check:
   - [ ] No em dashes
   - [ ] Pragmatic and direct
   - [ ] No corporate speak or buzzwords
   - [ ] Sounds human
3. If any check fails, rewrite before posting

### Step 5: Post the Content

**Reddit (via browser):**
1. Navigate to the thread
2. Post the comment using the RickClaw_Dev account
3. Note the thread URL and subreddit

**Discord (via message tool):**
1. Send to the appropriate channel
2. Keep formatting Discord-friendly (no tables, use bullet lists)

**Other forums:**
1. Use browser to navigate and post
2. Follow platform-specific guidelines

### Step 6: Track and Follow Up

1. Log the outreach in today's memory file:
   ```
   ## Outreach Log
   - Platform: Reddit
   - Subreddit: r/example
   - Thread: [title](url)
   - Type: comment
   - Topic: AI agents for small business
   - Posted: 2024-01-15 14:30 EST
   - Follow-up needed: Yes/No
   ```
2. For threads needing follow-up:
   - Check back within 24 hours for replies
   - Respond to questions or engagement
   - Do NOT get into arguments. Disengage gracefully if needed
3. Track karma and engagement metrics weekly

## Proactive Post Guidelines

When creating original posts (not responses):

### Reddit Posts
1. Choose the right subreddit from references/reddit-strategy.md
2. Post types that work well:
   - "How we solved X" (experience sharing)
   - "Lessons learned building Y" (retrospective)
   - "What tools do you use for Z?" (discussion starter)
   - Resource or guide sharing (if genuinely useful)
3. Format for Reddit:
   - Descriptive title (no clickbait)
   - TL;DR at the top for long posts
   - Use Reddit markdown
   - Flair appropriately

### Discord Posts
1. Share in appropriate channels (show-and-tell, projects, general)
2. Keep initial message short
3. Be ready to engage in the thread

## Rules of Engagement

1. **Never spam.** One post per subreddit per day max
2. **Build karma legitimately.** Help people. Answer questions. Be useful
3. **No astroturfing.** Be transparent about who you are
4. **Respect community rules.** Read sidebar/rules before posting
5. **Disengage from trolls.** Do not argue. Move on
6. **Track everything.** Log all outreach for pattern analysis
7. **Quality over quantity.** One great comment beats ten mediocre ones

## Karma Building Strategy

For new or low-karma accounts:
1. Start by commenting helpfully on others' posts (no self-promotion)
2. Answer technical questions in your areas of expertise
3. Share useful resources (not your own) that help people
4. Participate in weekly threads and AMAs
5. Build reputation for 2-4 weeks before any promotional content
6. Aim for 100+ karma before posting original content with links
