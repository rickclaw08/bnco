```
CALCULATIONS
- Monthly Net Benefit: $___  (Total Gain - Monthly Cost)

- Probability of success: ___%
```


## How to Use These Templates

1. Copy the relevant template
2. Fill in known values
3. Use formulas from SKILL.md for calculations
4. Run sensitivity analysis (bear/base/bull) for major decisions
5. Present to Rick with bottom-line first
6. Save completed analysis to memory files
