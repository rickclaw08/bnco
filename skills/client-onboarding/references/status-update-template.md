## Template Usage Notes

- Always send on the agreed day, even if progress was minimal
- Be honest about delays. Clients appreciate transparency
- If status is yellow or red, lead with the problem and the plan to fix it
- Keep the email scannable (bullet points, bold for key items)
- No em dashes
- If there are no blockers, say "No current blockers" (do not skip the section)
- Include client action items prominently so they know what they need to do
